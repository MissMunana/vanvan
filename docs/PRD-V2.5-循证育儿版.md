## 核心功能清单

### 1. 循证育儿知识库

#### 1.1 设计原则

- **循证为本**：所有内容基于高质量研究证据（系统综述、RCT、权威指南）
- **标注来源**：每篇内容标注参考来源（如AAP、WHO、Cochrane等）
- **避免焦虑**：传递信息而非制造焦虑，语气温和务实
- **实用导向**：提供可操作的建议，而非泛泛的理论

#### 1.2 内容分类

**分龄育儿指南**
```
0-1岁：婴儿期
├── 母乳喂养与配方奶指南
├── 辅食添加时间与方法（WHO建议6月龄引入）
├── 睡眠安全（仰卧睡姿、SIDS预防）
├── 疫苗接种时间表
└── 常见问题：肠绞痛、湿疹、吐奶

1-3岁：幼儿期
├── 语言发展与促进策略
├── 如何应对"可怕的两岁"（Terrible Twos）
├── 如厕训练的时机与方法
├── 饮食习惯培养
└── 屏幕时间指南（AAP建议）

3-6岁：学龄前期
├── 幼儿园适应与分离焦虑
├── 社交技能培养
├── 情绪管理引导
├── 阅读习惯启蒙
└── 运动与身体发育

6-12岁：学龄期
├── 学习习惯与自主性培养
├── 同伴关系与社交
├── 自信心与抗挫力培养
├── 青春期前期准备
└── 网络安全与数字素养
```

**行为管理策略（基于循证方法）**

**正面管教核心策略（Jane Nelsen）**
```
1. 自然后果与逻辑后果
   ├── 定义：让孩子体验行为的自然结果
   ├── 示例：不吃晚饭 → 到下一餐前会饿（自然后果）
   ├── 示例：乱丢玩具 → 玩具暂时"休息"一天（逻辑后果）
   └── 原则：后果需与行为相关、尊重、合理、预先告知

2. "和善而坚定"原则
   ├── 和善：尊重孩子的感受和需求
   ├── 坚定：坚守规则和底线
   └── 示例："我理解你很想继续玩（和善），但现在是睡觉时间了（坚定）"

3. 鼓励 vs 表扬
   ├── 鼓励关注过程和努力："你花了很多时间练习，进步很大！"
   ├── 表扬关注结果和天赋："你真聪明！"（应减少）
   └── 依据：Carol Dweck成长型思维研究
```

**情绪辅导五步法（John Gottman）**
```
1. 觉察孩子的情绪
2. 将情绪时刻视为亲近和教导的机会
3. 带着共情倾听和认可孩子的感受
4. 帮助孩子用语言标记情绪
5. 在设定界限的同时帮助孩子解决问题
```

**常见疾病家庭护理指南**
```
发烧护理
├── 何时需要退烧药（基于症状而非单纯体温）
├── 物理降温的正确做法与误区
├── 何时必须就医
└── 补液：少量多次饮水/口服补液盐

腹泻护理
├── 脱水评估
├── 口服补液盐（ORS）使用
├── 饮食管理：继续正常饮食，不需要禁食
└── 何时就医

咳嗽护理
├── 咳嗽是保护性反射，不必急于止咳
├── 1岁以上可尝试蜂蜜缓解
├── 保持空气湿润
└── 何时就医

过敏管理
├── 常见过敏原识别
├── 过敏急性发作处理
└── 日常预防措施
```

#### 1.3 内容呈现方式

**卡片式知识**
- 核心结论（大字号）
- 简要解释（2-3段）
- 参考来源（标注可信度）
- 相关阅读链接

**情景化推送**
- 根据孩子年龄和当前场景推送相关知识
- 例：孩子6月龄时推送辅食添加指南
- 例：记录发烧时推送发烧护理知识

**互动式问答**
- 常见问题Q&A形式
- 快速搜索功能

**科学辟谣专区**
- 针对常见育儿谣言进行循证辟谣
- 例："捂汗能退烧吗？"
- 例："退热贴有用吗？"

#### 1.4 参考来源标注规范

| 来源级别 | 示例 | 可信度标签 |
|---------|------|----------|
| 系统综述/Meta分析 | Cochrane Review | ⭐⭐⭐⭐⭐ |
| 权威指南 | AAP / WHO / 中华医学会指南 | ⭐⭐⭐⭐⭐ |
| RCT研究 | 发表于NEJM / Lancet / Pediatrics | ⭐⭐⭐⭐ |
| 专家共识 | 中国专家共识 | ⭐⭐⭐ |

### 2. 情绪与心理发展模块

#### 2.1 设计背景

**循证依据**
- 儿童情绪能力是预测未来学业表现、社交成功和心理健康的重要指标
- 能用语言标记情绪的儿童，行为问题更少、情绪调节能力更强
- 早期识别情绪行为问题有助于及时干预

**设计原则**
- 这不是心理诊断工具，而是促进亲子沟通的辅助工具
- 语气温暖、去病理化，不给孩子"贴标签"
- 鼓励家长将情绪时刻视为教育机会

#### 2.2 儿童情绪日记（Mood Check-in）

**功能描述**
每天一次简单的情绪记录（建议在睡前完成），帮助孩子建立情绪觉察能力。

**分龄交互设计**

**3-5岁版本（极简）**
```
"今天你的心情像哪个小动物？"
├── 🐱 开心猫咪（快乐）
├── 😿 哭哭小猫（难过）
├── 🐯 生气老虎（愤怒）
├── 🐰 害怕小兔（恐惧）
└── 🐻 平平的小熊（平静）

选择后：
- 吉祥物回应："谢谢你告诉我！开心猫咪的一天真棒！"
- 如选择负面情绪："抱抱你，每个人都会有不开心的时候哦"
- 不追问原因（低龄儿童难以归因）
```

**6-8岁版本（可选原因）**
```
"今天你的心情怎么样？"
├── 😄 很开心
├── 🙂 还不错
├── 😐 一般般
├── 😟 有点不开心
└── 😢 很难过 / 😠 很生气

选择后可选填："是什么让你这样呢？"
├── 和朋友玩
├── 被表扬了
├── 和朋友吵架了
├── 作业太多了
└── 其他：______
```

**9-12岁版本（丰富情绪词汇）**
```
"用一个词描述今天的心情？"

情绪词汇轮盘（可旋转选择）：
├── 快乐家族：开心/兴奋/满足/自豪/感激/放松
├── 悲伤家族：难过/失落/孤独/失望/委屈
├── 愤怒家族：生气/烦躁/不公平/嫉妒
├── 恐惧家族：害怕/紧张/焦虑/担心
└── 平静家族：平静/无聊/累了

可选写一句话日记
```

**数据可视化（家长端）**
- 月度情绪分布饼图
- 情绪趋势折线图
- 负面情绪关联因素分析
- **注意**：只在家长端显示，儿童端不显示

#### 2.3 亲子冲突复盘工具

**场景**：家长和孩子发生冲突后，提供结构化的复盘框架

**冲突复盘三步法（基于John Gottman情绪辅导理论）**

```
第一步：冷静（Calm Down）
├── 系统提示：建议等到双方都冷静后再复盘（至少15分钟）
├── 家长自检："我现在能平静地和孩子对话吗？"
└── 如果还在生气：提供家长自我调节建议

第二步：连接（Connect）
├── 引导家长和孩子一起回顾：
│   ├── "发生了什么事？"（让孩子先说，不打断）
│   ├── "当时你的感觉是什么？"（认可情绪）
│   ├── "你觉得爸爸/妈妈当时的感觉是什么？"（培养共情）
│   └── "爸爸/妈妈当时的感觉是______"
└── 目标：双方都感到被理解

第三步：解决（Solve）
├── "下次遇到这种情况，我们可以怎么做？"
├── 一起制定1-2条具体的约定
├── 记录在App中，下次类似情境触发时提醒
└── 约定示例："下次不想做作业时，先告诉妈妈'我需要休息10分钟'"
```

**记录与追踪**
- 冲突频率趋势（是否逐月减少）
- 高频冲突场景识别
- 约定执行情况跟踪

### 3. 睡眠管理模块

#### 3.1 功能概述

帮助家长建立科学的儿童睡眠习惯，改善睡眠质量。

#### 3.2 睡眠记录

**数据录入**
```javascript
// 睡眠记录表 (sleep_records)
{
  recordId: String,
  childId: String,
  date: Date,
  bedTime: Time,            // 上床时间
  sleepTime: Time,          // 入睡时间
  wakeTime: Time,           // 醒来时间
  getUpTime: Time,          // 起床时间
  duration: Number,         // 总睡眠时长（分钟）
  napTimes: [{              // 午睡记录
    start: Time,
    end: Time,
    duration: Number
  }],
  sleepQuality: String,     // 很好/一般/较差
  notes: String,            // 夜醒次数、难以入睡等
  createdAt: Date
}
```

**睡眠统计**
- 平均入睡时间
- 平均睡眠时长
- 睡眠质量趋势
- 与推荐睡眠时长对比

#### 3.3 分龄睡眠建议（基于AAP和NSF推荐）

| 年龄段 | 推荐睡眠时长（24小时） | 建议作息 |
|-------|---------------------|---------|
| 0-3月 | 14-17小时 | 无固定规律，按需喂养 |
| 4-11月 | 12-15小时 | 夜间连续睡眠增加，2-3次小睡 |
| 1-2岁 | 11-14小时 | 夜间11-12小时，1-2次小睡 |
| 3-5岁 | 10-13小时 | 夜间10-11小时，1次午睡或不午睡 |
| 6-12岁 | 9-12小时 | 夜间连续睡眠，通常不午睡 |

#### 3.4 睡前仪式流程构建器

**功能描述**
帮助家长建立固定的睡前仪式，提高睡眠质量。

**睡前仪式模板**
```
睡前30分钟开始：
├── 1. 洗澡（10分钟）
├── 2. 换睡衣、刷牙（5分钟）
├── 3. 调暗灯光
├── 4. 讲睡前故事（10分钟）
├── 5. 温柔拥抱，说晚安
└── 6. 熄灯，播放白噪音/轻音乐（可选）
```

**可自定义流程**
- 家长可添加/删除/调整步骤
- 设置每个步骤的时长
- 设置提醒时间
- 完成步骤打勾（培养孩子参与感）

### 4. 应急安全模块

#### 4.1 设计背景

意外伤害是中国1-14岁儿童的首位死亡原因。家长在紧急情况下往往慌乱，需要能在几秒内获取关键信息的工具。

#### 4.2 紧急联系卡

**功能描述**
一张包含孩子所有关键信息的电子急救卡，支持一键调用和锁屏展示。

```
┌──────────────────────────────────────┐
│  🆘 紧急联系卡                        │
│                                        │
│  姓名：______    血型：______          │
│  出生日期：______    体重：____kg       │
│                                        │
│  ⚠️ 过敏史：                           │
│  药物：______                          │
│  食物：______                          │
│                                        │
│  📞 紧急联系人：                       │
│  爸爸：138xxxx（一键拨打）              │
│  妈妈：139xxxx（一键拨打）              │
│                                        │
│  🏥 常去医院：______                   │
│                                        │
│  急救电话：120    中毒热线：010-83132345│
└──────────────────────────────────────┘
```

**使用场景**
- 手机锁屏快捷方式
- 打印后放在孩子书包里
- 外出旅游时随身携带
- 支持一键拨打急救电话和紧急联系人

#### 4.3 分龄急救速查

**常见意外伤害快速处理指南**

```
🩹 外伤出血
├── 按压止血（5-10分钟）
├── 清水冲洗伤口
├── 消毒、包扎
└── 何时就医：出血不止、伤口深、动物咬伤

🔥 烧烫伤
├── 立即冷水冲洗（15-30分钟）
├── 不要涂抹牙膏、酱油等
├── 轻度烫伤可涂烫伤药膏
└── 何时就医：面积大、起水泡、烧伤深

😵 头部撞击
├── 冰敷（减少肿胀）
├── 观察精神状态
└── 立即就医：持续呕吐、意识不清、抽搐

🤢 误食异物
├── 不要催吐（可能二次伤害）
├── 记录误食物品名称和时间
└── 立即拨打中毒热线：010-83132345
```

#### 4.4 意外伤害预防清单

**分龄安全检查清单（可打印贴在冰箱上）**

**0-1岁**
- [ ] 床栏高度足够，间距合理
- [ ] 避免同床睡眠（SIDS风险）
- [ ] 玩具无小零件（窒息风险）
- [ ] 洗澡水温<40℃

**1-3岁**
- [ ] 插座安装保护盖
- [ ] 危险物品（药品、清洁剂）高处存放
- [ ] 窗户安装防护栏
- [ ] 桌角、柜角安装防撞角

**3-6岁**
- [ ] 交通安全教育（红绿灯、斑马线）
- [ ] 游泳必须成人监护
- [ ] 不独自在家
- [ ] 教会紧急情况拨打120

**6-12岁**
- [ ] 网络安全教育
- [ ] 独自外出告知家长去向
- [ ] 学会基础自救技能
- [ ] 防性侵教育

### 6. 家长学堂

#### 6.1 功能描述

为家长提供系统的育儿知识学习路径。

#### 6.2 课程体系

**新手父母必修课**
- 第1课：了解孩子的发展规律
- 第2课：正面管教的核心理念
- 第3课：有效沟通与情绪辅导
- 第4课：如何设置合理的规则和界限
- 第5课：培养孩子的内在动机

**分龄专题课**
- 婴儿期专题：睡眠训练、辅食添加、发育追踪
- 幼儿期专题：语言启蒙、如厕训练、情绪管理
- 学龄前专题：幼小衔接、社交技能、阅读启蒙
- 学龄期专题：学习习惯、时间管理、青春期准备

**常见问题解决**
- 孩子不爱吃饭怎么办？
- 孩子发脾气怎么办？
- 孩子不爱分享怎么办？
- 孩子撒谎怎么办？

#### 6.3 呈现方式

- 短视频（3-5分钟）
- 图文文章（配插画）
- 音频课程（开车时听）
- 互动测试（了解自己的育儿风格）
