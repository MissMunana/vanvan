## 核心功能清单

### 1. 拖拽式加分交互

#### 1.1 交互设计

**场景：拖动任务到积分池**

```
交互流程：
1. 用户长按任务卡片（>500ms）
   ├── 震动反馈（10ms）
   ├── 卡片放大至1.2倍
   └── 出现阴影效果

2. 积分池高亮闪烁
   └── 提示"松手即可加分"

3. 拖动过程中
   ├── 卡片跟随手指移动
   ├── 移至积分池范围内：积分池变色（绿色高亮）
   └── 移出积分池范围：积分池恢复常态

4. 释放手指
   ├── 在积分池内：执行加分动画
   └── 在积分池外：卡片弹回原位（弹簧效果）
```

**加分动画时间轴**

```
0ms:    任务卡片开始变形（卡片→星星）
200ms:  星星飞向积分池（贝塞尔曲线路径）
400ms:  星星进入积分池，爆开粒子效果
500ms:  积分数字开始增长动画（0.5秒滚动）
600ms:  播放音效（叮咚声）
1000ms: 积分池发光效果（淡入淡出）
1200ms: 任务卡片标记为"已完成"
```

#### 1.2 可用性优化

**低龄儿童适配（3-5岁）**
- 拖拽作为"高级交互"，默认点击+确认
- 拖拽目标区域加大（屏幕上半部分都算命中）
- 增加吸附效果：拖到接近积分池时自动吸附
- 首次使用提供手把手引导教程

**手机适配**
- 小屏手机（<6英寸）：积分池占屏幕上1/3
- 大屏手机：积分池适当缩小，避免误触

### 3. 习惯毕业机制

**理论基础**：习惯养成平均需要66天，分阶段设计激励策略。

#### 3.1 习惯四阶段

```
阶段一：启动期（1-7天）
├── 目标：降低启动阻力，建立初始动力
├── 激励：高积分奖励（基础积分 × 1.5）
├── 反馈：每天完成都有特殊鼓励动画
├── 提示：吉祥物每天提醒和鼓励
└── 文案："今天是第X天，你已经很棒了！"

阶段二：坚持期（8-21天）
├── 目标：建立行为模式，强化自动化
├── 激励：积分恢复正常值
├── 视觉：引入连续打卡视觉激励（如种树成长）
├── 里程碑：每7天一次"小里程碑"庆祝
└── 文案："坚持第X天，习惯的力量正在积累！"

阶段三：稳定期（22-66天）
├── 目标：从外在激励过渡到内在动机
├── 激励：积分逐步递减（基础积分 × 0.8）
├── 反思：引入内在反思（"做完这件事你感觉怎么样？"）
├── 荣誉：荣誉称号替代积分激励
└── 文案："你已经是习惯小达人了！"

阶段四：毕业期（66天+）
├── 目标：习惯内化，自然而然
├── 激励：无积分奖励，保留打卡记录
├── 仪式：获得永久"习惯大师"勋章
├── 动画：习惯毕业典礼动画
└── 引导：鼓励孩子帮助弟弟妹妹建立同样的习惯
```

#### 3.2 习惯毕业典礼

**触发条件**：某习惯连续完成66天

**仪式流程**
```
1. 弹出全屏庆祝动画
   ├── 烟花效果
   ├── 吉祥物鼓掌
   └── 文案："恭喜你！[习惯名称]已经成为你的一部分了！"

2. 颁发"习惯大师"勋章
   ├── 勋章飞入成就墙
   ├── 播放胜利号角
   └── 震动反馈

3. 成长回忆（可选）
   ├── 回顾这66天的历程
   ├── 展示完成记录时间线
   └── 可保存为图片分享

4. 鼓励传递（可选）
   └── "你愿意帮助其他小朋友也养成这个习惯吗？"
```

#### 3.3 视觉化进度

**习惯成长树**
- 第1天：种下种子
- 第7天：发芽
- 第21天：长出叶子
- 第66天：开花结果

**习惯拼图**
- 每完成一次任务，拼图增加一块
- 66块拼图完成后，显示完整图案

### 4. 勋章系统

#### 4.1 勋章分类

**习惯勋章**
- 🌱 启动者（第1次完成任务）
- 🔥 坚持者（连续7天）
- ⭐ 习惯达人（连续30天）
- 🏆 习惯大师（连续66天，习惯毕业）

**积分勋章**
- 💰 小富翁（累计获得100积分）
- 💎 大富翁（累计获得500积分）
- 👑 积分王（累计获得1000积分）

**特殊勋章**
- 🎯 全能选手（5个不同类型任务都完成过）
- 🌟 全勤之星（连续7天每天完成任务）
- 🚀 进步之星（本周完成任务数比上周多50%）

#### 4.2 勋章墙

**展示设计**
- 网格展示获得的勋章（3D卡片效果）
- 未获得的勋章显示为剪影
- 点击勋章查看获得条件和时间
- 勋章总数显示

**互动效果**
- 新获得的勋章高亮闪烁
- 点击勋章播放获得时的动画
- 可设置"最喜欢的勋章"展示在首页
